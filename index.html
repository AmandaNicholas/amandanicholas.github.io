About Me

Hi, I'm Amanda. I've been a Technical Writer in the software industry since 2009. This site is my online portfolio and its purpose is to showcase the work I've done over the past few years. If you'd like to know more, please email me and I'll be happy to send you my full CV.

Recent Work

Quick Start Guides for Outdoor Payment Terminals

Invenco needed printed documentation to put in the box alongside their Outdoor Payment Terminal devices bound for their clients' developers. Working with the sales and engineering teams, I came up with short, easily understandable documents for each device type.

User Guide for Cloud Portal to Manage Invenco Outdoor Payment Terminals

I created the User Guide for this product, which encompassed several user roles and more than 70 tasks associated with performing remote management of Invenco-made outdoor payment terminals.

API Developer Guides

I created two RESTful API documents for Invenco products: one for media companies who wished to send information to be displayed on Outdoor Payment Terminals, and another for mobile providers who wished to authorise fuel pumps on a gas station forecourt.

Quick Start Guide for Fleet Management Application

Coretex needed a Quick Start Guide for their fleet-management application to hand to a major new client. I had already written 40+ new task-based articles for their online knowledge base, so I picked the most important tasks for the Quick Start Guide, referring to the knowledge base where appropriate.

E-newsletters for Meadowbank and St Johns Residents Association

In 2014 I created the website for Meadowbank and St Johns Residents Association and started sending monthly newsletters about local topics of interest (my final issue sent June 2017).
